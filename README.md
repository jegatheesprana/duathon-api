### Technology Stack
	Node.js
    React
    MongoDB

#### Instructions for running
	1. Install the dependencies
	npm install

    2. Connet MongoDB 
    Add the MongoDB connection url in .env file

	3. Run Hot-reload Server
	npm run dev

	4. API Server will be usually available at,
	http://localhost:8000
